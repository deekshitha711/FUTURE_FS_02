# FUTURE_FS_02
# 📊 Client Lead Management System (Mini CRM)

A simple full-stack CRM application to manage client leads. This project allows users to add, view, update, and delete leads efficiently.

---

## 📌 Project Overview

This CRM system helps in managing client information collected through forms. It supports tracking lead status and maintaining notes for follow-ups.

---

## ✨ Features

- ➕ Add new leads (Name, Email, Source, Notes)
- 📋 View all leads in table format
- 🔄 Update lead status (New / Contacted / Converted)
- 🗑 Delete leads
- 🔍 Search leads by name
- 🎯 Filter leads by status
- 🔐 Simple admin login

---

## 🛠 Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Local using MongoDB Compass)
